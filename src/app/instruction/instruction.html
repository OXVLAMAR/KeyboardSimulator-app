Профиль:

В профиле Вы можете изменить уровень сложности и свои учетные данные. Для изменения уровня сложности переключите маркер на вкладке «Профиль». Для изменения учетных данных нажмите кнопку «Изменить данные», а затем введите новые данные в окне «Изменить данные»

Упражнения:

Для просмотра доступных на выбранном уровне сложности упражнений перейдите на вкладку «Выбор упражнения». В ней Вы можете выбрать упражнение для прохождения. Для этого выберите упражнение и система отобразит его текст, количество допустимых ошибок, скорость прохождения упражнения и время. Чтобы начать прохождение упражнения, нажмите кнопку «Начать». Прохождение упражнения можно завершить. Для этого нажмите кнопку «Завершить», система выдаст сообщение о текущей статистике по данному упражнению.

Статистика:

Чтобы узнать результаты всех успешно пройденных упражнений, нажмите на вкладку «Статистика». Страница, на которую Вы перейдете, содержит таблицу с датами выполнения упражнений, количеством допущенных ошибок, скоростью выполнения упражнения.
